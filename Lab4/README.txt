The user can give 1 integer input at a time.
If the stack is empty then the number is stored in the stack.
If the number given is greater than the top of the stack.

then the number is squashed, this repeats until the
numbers in the stack cannot be squashed anymore.

If the number given is smaller then it is added to the
top of the stack.

##########

make and change directory to bin and run using ./main
make clean - to clean object and binary files

##########

No optional features

##########

Only 1 input is taken in at a time and it only
works for integers and does not work for any other
data type.
